# Startup Program Kit — Pitch

## One-liner
Vercel is the fastest path from idea → deployed product → iteration.

## What a Startups SE is selling
Not “hosting.”

You’re selling:
- Speed to first deploy
- Developer velocity
- Low operational burden
- A platform that scales with the company

## Common startup objections + responses

### “We can host this anywhere.”
You can. The point is time-to-value.
- Previews for every PR
- Built-in Git-based deploy workflow
- Integrated primitives for the modern web (edge, serverless, AI)

### “We don’t want DevOps overhead yet.”
Great — don’t take on unnecessary ops.
- Start serverless on Vercel
- Use managed services like Postgres/auth
- Add observability early without a platform rewrite

### “Will this scale?”
It’s designed to scale without replatforming.
- Add caching, edge behavior, and performance instrumentation as you grow
- Keep the workflow consistent from MVP to production

## Where this repo fits
This project is a reference implementation you can use in:
- Startup onboarding calls
- Workshops / webinars
- Accelerator office hours

It demonstrates a realistic MVP shape and a clean path to production hardening.
