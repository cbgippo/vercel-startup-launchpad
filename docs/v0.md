# v0 Workflow

This repo is compatible with v0 as a UI iteration accelerator.

## How to use v0 with this project

1) Start with a page-level prompt
- Describe the page and components you want.
- Include existing routes and what data is available.

2) Keep a consistent design system
- Reuse the existing Tailwind tokens and layout patterns.

3) Prefer small diffs
- Ask v0 for one page/section at a time.

## Suggested v0 prompts for this repo

### Onboarding wizard refinement
"Design a modern onboarding wizard page for a startup SaaS with 3 steps: create brief, generate assets, production checklist. Use Tailwind. Include progress state and template cards."

### Brief detail page polish
"Improve the brief detail page UI: two-column layout on desktop, left notes, right generated assets. Add better typography and a regenerate button." 

### Dashboard improvements
"Add a dashboard empty state with a clear primary CTA and secondary link to onboarding. Use a simple card layout." 

## Where to apply v0 output
- `src/app/dashboard/onboarding/page.tsx`
- `src/app/dashboard/page.tsx`
- `src/app/dashboard/briefs/[id]/page.tsx`
